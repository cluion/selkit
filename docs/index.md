features:
  - title: Truly headless
    details: A pure TypeScript state machine with zero DOM and zero framework dependencies. Behavior is written once; adapters only render state.
  - title: One core, three adapters
    details: The same @selkit/core powers @selkit/dom, @selkit/vue and @selkit/react with a unified API. Learn it once, use it anywhere.
  - title: No jQuery, zero runtime deps
    details: A drop-in alternative to select2 and Tom Select without the jQuery baggage. The default positioner ships in-core.
  - title: Batteries included
    details: Search (substring, diacritics-insensitive, fuzzy), multiple select, tagging, async loading with pagination, virtual scrolling, forms and RTL.
  - title: Accessible by default
    details: ARIA combobox/listbox roles, active-descendant management and full keyboard navigation come from the core.
  - title: Themeable
    details: A neutral base theme plus a scoped Bootstrap 5 theme, both driven by CSS custom properties. Restyle with a few variables.
