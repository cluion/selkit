Selkit — Vanilla Demo

純原生 JS，無 jQuery、無框架。大腦是 @selkit/core，渲染是 @selkit/dom。

試試輸入過濾、↑↓ 鍵、Enter 選取、Esc 關閉。
可連續選取、tag 可移除、空輸入時 Backspace 刪除最後一個。
同一份元件，容器加上 selkit-theme-bs5 class 即融入 BS5 外觀。
輸入 blbr 也能搜到 Blueberry（子序列比對）。
多選；輸入時輸入框寬度貼齊文字而非撐滿。
控制項窄；下拉仍展開到最長選項的寬度，不被截斷。
輸入 a：以 a 開頭的選項排到最前。
繁中空狀態訊息：剛開啟顯示「再輸入 N 字」，輸滿後若無相符顯示「查無資料」。
已選項留在清單並打勾；再點一次即取消。
選幾個後，把輸入框清空再按 Backspace：最後一個 tag 被刪、其文字回到輸入框可編輯。
輸入或貼上以逗號／空白分隔的字串，逗號前的片段自動成為 tag。
輸入不存在的字，清單末尾出現可點的建立列；此處用自訂文字「建立 …」。
下拉每個選項前加 🍎；已選項顯示 ✓。回傳 Node 即可放任意標記。
每個 tag 前面加上 🔖；回傳 Node 即可放任意標記。
箭頭換成 ▸/▾、清除鈕與移除鈕換成 emoji、分組標題加前綴、查無資料時顯示自訂文案。
已選項從清單隱藏；拖曳 tag 可重新排序。
捲到下拉底部自動載入下一頁（共 5 頁）。
1000 筆選項，只渲染可視切片。
整個區塊設 dir="rtl"，tag 與箭頭自動翻轉。
